# Project Name
DigiTools

## Description
DigiTools is a modern and responsive web application designed to provide users with powerful digital tools in one place. It offers a clean UI, smooth user experience, and efficient performance for managing products and interactions.

---

##  Technologies Used
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- React-Toastify (NPM Package)
- JSON (for product data)

---

## Features
-  Add to Cart functionality with dynamic cart count  
-  Mobile Navigation Menu – Hamburger menu for small devices   
-  Fast and smooth UI with modern styling
